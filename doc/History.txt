История версий Ru.Board EasyWrite
----------------------------------
Copyright © 2006-2007 GalaxyWorks

Версия 0.3.3a (2007-05-18)
-----------------------------
* Включена поддержка unicode в Preview/Preview in browser
+ Добавлена работа с шаблонами сообщений
- Восстановлена совместимость с Windows 98
- Ошибки при отрисовке панели состояния


Версия 0.3.2 (2006-03-15)
-----------------------------
* Улучшена отрисовка некоторых кнопок при отключенных темах Windows
* Убрана дублирующая заголовок панели надпись "Предпросмотр поста"
* Исправлена работа со шрифтами в младших версиях Windows
* Убран лишний слэш ("\") перед апострофом и слэшом
* Убран "модераторский плюсомет" из коллекции смайлов Ru-Board Extra
* Правильное декорирование ссылок
* Убран мусор при нажатии Del в конце строки
* Правильные коды непоименованных цветов
+ Подсказки с адресами к ссылкам
+ Поддержка тегов sub и sup
+ Поддержка тега [#]
+ Сохранение второстепенных настроек (параметры диалогов Спец. функций)
+ Вертикальная полоса прокрутки сохраняет положение при изменении сообщения
+ Просмотр сообщения в браузере
- Параметры диалогов не сохраняются в Настройки по умолчанию


Версия 0.3.1 (2006-01-26)
-----------------------------
* Исправлен вызов смайлов на локализованных версиях Windows
+ Загрузка настроек по умолчанию
+ Опция для употребления коротких тэгов
+ Опция – снимать ли выделение текста после применения тэга


Версия 0.3 beta (2006-01-22)
-----------------------------
* Воспроизведена функциональность версии 0.2 beta
+ С целью учета количества пользователей введена бесплатная регистрация 
+ Поддержка unicode
+ Предпросмотр сообщения теперь встроен в программу
+ Авто-сохранение набираемого текста (файл message.txt)
+ Многоязыковая поддержка
+ Анимация смайлов (gif-файлы)
+ Настраиваемые коллекции смайлов (см. папку smiles), css стиль (см. папку style) и шаблон предпросмотра (см. папку preview)
+ Основная коллекция смайлов использует тэги форума
+ Тэг цвета текста использует названия цвета только для официальных цветов w3.org (остальные - шестнадцатеричные)
+ Текст остается выделенным после применения обрамляющих тэгов
+ Начата работа по поддержке нескольких форумов
+ Более гибкие редакторы таблиц, списков и цвета
+ Сохранение сообщения
+ Специальные функции для защиты от спама и вставки нескольких ссылок
+ Восстановление последнего расположения окна/панелей и т.д. при загрузке
+ Контекстное меню при сворачивании окна в системную область
- Нет функции транслитерирования текста
- Коллекция дополнительных смайлов использует старую разбивку по категориям
- Нет настройки шрифта редактора
- Использован старый шаблон предпросмотра, воспринимающий не все тэги
- Неименованные цвета текста в предпросмотре и на форуме отображаются по-разному
